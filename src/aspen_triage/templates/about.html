About aspen-triage

AI-powered triage for the Aspen Discovery JIRA

What is this?

aspen-triage harvests issues from Aspen Discovery's JIRA and makes them searchable using semantic AI. Describe a problem in plain language and find related issues — even with different terminology.

Built by ByWater Solutions to help the Aspen community triage and fix issues faster.

Features

S
Semantic search

Find issues by meaning, not keywords. Enter a DIS-XXXX key to jump straight to an issue.

V
AI verdicts

Each search result gets a Claude-generated verdict: duplicate, already in progress, resolved, etc.

G
Issue grouping

Manually cluster related issues into groups for coordinated triage.

R
AI recommendations

Per-issue fix recommendations using Aspen's security + QA guidelines.

P
GitHub PR creation

Generate code fixes and one-click create Draft PRs on Aspen-Discovery/aspen-discovery with the JIRA key in the title.

Workflow

  1. Find — Search or browse JIRA issues
  2. Recommend — Claude reads the issue + Aspen guidelines and produces a fix plan
  3. Generate — Claude writes the actual code fix, shown as a diff
  4. PR — One-click Draft PR on GitHub (branch name and title reference the DIS key)
  5. Review — Human review is always required before merge

Technology

Python, FastAPI, SQLite, fastembed (BAAI/bge-small-en-v1.5), Claude by Anthropic. Source on GitHub.

AI-generated PRs include a note in the body; Claude's contribution is disclosed.
