How to get tokens

GitHub

  1. Go to GitHub → Settings → Fine-grained PATs
  2. Generate new token, name it aspen-triage
  3. Repo access: "All repositories"
  4. Permissions: Contents: Read and write, Pull requests: Read and write

JIRA

  1. Go to id.atlassian.com/manage-profile/security/api-tokens
  2. Create API token
  3. Paste above, along with the email you use to sign into JIRA
